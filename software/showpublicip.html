ShowPublic IP

Instantly display your public IP address and location with a single click.

While command-line tools are a staple for many power users, having a graphical interface to instantly retrieve your public IP and geolocation data is simply more convenient. ShowPublic IP offers a streamlined solution that allows users to view their public IP address, geographical location, and network provider details with ease.

The application displays the current IP address, city, region, and country, along with precise geographical coordinates. Additionally, it provides the name of the Internet Service Provider (ISP) and your system’s time zone.

Each data field comes with a dedicated Copy button, so you can instantly send the information to your clipboard for quick use. Need to check if your VPN is masking your IP properly? Simply press the Refresh button and ShowPublic IP will instantly recheck and display updated data.

This tool is ideal for users who need quick access to public IP data for security audits, remote configuration, VPN testing, or technical diagnostics — all from a lightweight desktop utility.
