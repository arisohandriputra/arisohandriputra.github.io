HeatMon is a lightweight CPU monitoring tool. This documentation describes its menus, options, and usage instructions.

Main Window
- Displays CPU name and number of cores
- Shows temperature per core (°C or °F)
- Shows voltage per core (mV or V)
- Real-time chart with legends
- Fan speed indicator (RPM) if detected
Menu and Options
[ File Menu ]
  * Save Report (.txt)  - Export CPU stats
  * Save Graph (.png)   - Save chart image

[ View Menu ]
  * Graph Themes        - Choose one of 12 themes
  * Show Unlisted Core  - Display inactive cores
  * Stay on Top         - Keep HeatMon visible

[ Help Menu ]
  * Help Content (F1)
  * About - Program info
Controls & Settings
[ Temperature Units ]
- °C (default) or °F

[ Voltage Units ]
- mV (default) or V

[ Chart Themes ]
- Teal Default
- Slate Gray
- Dark Blue
- Midnight Purple
- Emerald Green
- Fire Orange
- Ocean Blue
- Steel Gray
- Sunset Pink
- Gold Brown
- Cyan Teal
- Silver
Data Export
- Save Report : File → Save Report or Toolbar button
- Save Graph  : File → Save Graph or Toolbar button
