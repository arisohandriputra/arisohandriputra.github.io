Welcome !

Hi! I’m Ari Sohandri Putra, a software developer from Indonesia. I’ve been coding for over 10 years. I got into programming when I inherited my older brother’s old computer running Windows XP. That’s when I started messing around with C++, Visual Basic, and Delphi, and even made my first game, a Tic Tac Toe ;). Since then, I’ve just kept learning and playing around with programming.

I’ve developed several software projects, including uDiscMounter (a CD/DVD image mounting tool), HardMon (a system performance monitoring app for CPU, GPU, RAM, HDD, and network), HeatMon (similar to HardMon but focused on temperature monitoring for CPU, GPU, and motherboard), uBattery (a battery monitoring app for laptops and UPS), and many more.

My Project : .

uDiscMounter
uDiscMounter is a tool that can manage virtual drives and mount disk image files with extensive format support on Windows.

HardMon
HardMon is a lightweight and efficient system monitor that displays real-time hardware stats in a clean, categorized layout.

HeatMon
HeatMon is accurate CPU temperature monitoring software that displays the temperature per core on the CPU.

OGG Decoder
OGG Decoder is a straightforward and reliable desktop application designed to convert OGG Vorbis audio files into standard WAV format.

ShowPublic IP
This small application can display the public IP address associated to your computer, as well as its location.

SandiBox
An offline-only password manager that provides users with an environment where they can safely store important, sensitive information.

uPDF Joiner
Simple, handy utility that gives users the possibility to split, merge, watermark, and protect their PDF files.

DNSwift
This minimalist application allows you to easily change DNS settings and switch between popular providers or set custom servers.

Program Blocker
Easy tool you can use to encrypt any executable files you wish, locking them with a password and thus preventing unauthorized access.

uBattery
uBattery is a real-time battery monitoring tool designed to offer detailed insights into your device's battery performance.
